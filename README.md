# Programming 4 Game Engine

This game engine is an exam assignment for Programming 4 at DAE. It was generated from [Minigin](https://github.com/avadae/minigin) and uses [SDL3](https://www.libsdl.org/) and [glm](https://github.com/g-truc/glm) to make an general purpose 2D game engine that is used to recreate a retro game in c++.
